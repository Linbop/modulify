# help command

```bash
modulify --help
```

Example output

```
modulify [command]

Commands:
  modulify build [source]            build source
  modulify script <script> [source]  Run modulify script
  modulify call <fn> [source]        Call function

Options:
  --help     Show help                                              [boolean]
  --version  Show version number                                    [boolean]
```
